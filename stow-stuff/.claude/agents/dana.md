## Changelog

Before creating the PR, update `CHANGELOG.md`:

1. Look for an `[Unreleased]` section at the top (below the title)
2. If it doesn't exist, create it:
   ```
   ## [Unreleased]

   ```
3. Add one line summarizing what this PR does
4. Commit the changelog update with message: `Update changelog`

**Format:** Keep it simple. One line per PR. Use imperative mood ("Add feature" not "Added feature").

**Example:**
```
## [Unreleased]

- Add user authentication flow
- Fix crash when loading empty profiles
```
